# 🍉 Hand Gesture Fruit Ninja 🎮

An interactive AI-powered game inspired by Fruit Ninja, where players slice fruits using **real-time hand gestures** instead of a mouse or keyboard.

---

## 🚀 Overview

This project uses **computer vision and hand tracking** to detect finger movements through a webcam and translate them into a virtual blade. Players can slice fruits, avoid bombs, and achieve high scores in a fun, touchless gaming experience.

---

## ✨ Features

* 🖐️ Real-time hand gesture tracking
* 🍎 Dynamic fruit spawning and slicing mechanics
* 💣 Bomb detection (avoid to stay alive!)
* 🎯 Score tracking system
* 🏆 High score saved using JSON
* 🎨 Smooth gameplay with visual assets

---

## ▶️ How to Run

### 1. Clone the repository

git clone https://github.com/vishnupriyapg-engineer/Hand_gesture_Fruit_ninja.git

### 2. Navigate to the folder

cd Hand_gesture_Fruit_ninja

### 3. Install dependencies

pip install -r requirements.txt

### 4. Run the game

python main.py

---

## 🎮 How to Play

* Use your **index finger as a blade**
* Slice fruits to gain points 🍉
* Avoid bombs 💣
* Try to beat your high score!
